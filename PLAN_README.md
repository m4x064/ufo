# Math Fit Yuri 計画README

`Math Fit Yuri` は、UFOゆーりの世界観で計算練習を少しずつ育てていく数学ゲームです。
ただ問題を解くだけでなく、途中再開、成長、ごほうび、復習診断を組み合わせて、毎日戻ってきたくなる教材にしていきます。

## 目標

- 小学生が迷わず遊べる入力と画面にする。
- PCではキーボード、スマホでは数字パッドを中心にする。
- 計算範囲をステージで区切り、次に何を練習しているか見えるようにする。
- 正解数、連続正解、時間、苦手分野を使って、復習しやすくする。
- ゆーりの成長とアイテム収集で、長く続けたくなる見た目にする。

## 現在の中心ループ

1. タイトルでミッションを選ぶ。
2. 問題に答える。
3. 正解・不正解を確認する。
4. Enterまたはボタンで次へ進む。
5. 結果画面で正答率、平均時間、得意・注意分野を見る。
6. 累計正解数に応じて、ゆーりLvとごほうびが増える。

途中で閉じても `localStorage` に保存し、タイトルの `つづきから` で再開できます。

## 実装済みステージ

- `Stage 1 一桁スター航路`: 一桁どうしの基礎。Stage 1-1 のたし算と Stage 1-2 のひき算は0を使わず、Stage 1-3 はかけ算とわり算を合わせる。
- `Stage 2-1 繰り上がりたし算航路`: 一桁どうしで答えが二桁になるたし算。
- `Stage 2-2 繰り下がりひき算航路`: 二桁から一桁をひいて、答えが一桁になるひき算。
- `Stage 2-3 繰り上がりかけ算航路`: 一桁どうしで答えが二桁になるかけ算。
- `Stage 2-4 繰り下がりわり算航路`: 二桁を一桁で割って、答えが一桁になるわり算。
- `Stage 3 二桁どうしひき算航路`: 十の位が違う二桁どうしのひき算で、答えが一桁。
- `Stage 4 二桁どうしわり算航路`: 二桁どうしのわり算で、答えが2〜9。
- `Stage 5 二桁ミックス航路`: 二桁と一桁、一桁と二桁の四則演算で、答えが二桁。
- `Stage 6 二桁ツイン航路`: 二桁どうしの四則演算で、答えが二桁。
- `Stage 7 三桁ジャンプ航路`: 三桁に上がる計算、三桁から二桁へ戻る計算。
- `Stage 8 同じ数かけ算航路`: 0〜100の二乗、0〜10の三乗、0〜10の四乗。
- `Test 1 Stage 1-5 復習診断`: Stage 1〜5からランダム10問、60秒制限。
- `四則演算ナビ`: 四則演算の入口確認。

Stage 8 は、式を `12 × 12` や `3 × 3 × 3` のように表示します。
指数表記ではなく、小学生がそのまま読める形を優先します。

## 出題設計

- 「全通り」が自然なステージは、問題プールを先に作ってシャッフルする。
- 診断系は短い問題数にして、苦手分野を見つけやすくする。
- 発展系は、現在のステージを復習診断にまとめてから次の単元へ進む。
- 式表記は、学校で見慣れた形を優先する。

## 成長とごほうび

累計正解数に応じて、1万問までゆーりLvとアイテムが増えます。
手に入れたアイテムはタイトル画面に散りばめ、ステーションが少しずつ豪華になります。

ごほうびの方針:

- 最初は小さな達成感を短い間隔で出す。
- 中盤は見た目が変わるアイテムを増やす。
- 後半はレア感のある装飾や称号にする。
- ただの報酬ではなく、「ゆーりが成長している」感じを残す。

## 画面方針

- PCでは一画面に収まり、左右のスペースを使う。
- スマホでは縦に見やすく、ダブルタップ拡大しにくくする。
- 答え欄は大きく、入力した数字を見間違えにくくする。
- 白い面を広げすぎず、背景とゆーり世界観を見せる。
- かわいいけれど、子どもっぽくしすぎない。

## BGM方針

- キラキラ系を主役にする。
- ノイズ系は似た印象が増えやすいので、集中用として少数だけ残す。
- アプリではループ用BGMだけを使う。
- まずは `Moon Glitter`、`Comet Dust`、`Orbit Music Box`、`Snow Starfall`、`Starlight Chimes`、`Crystal Focus` を中心にする。
- ノイズ系は `Focus Room` と `Soft Rain` を残す。

## 入力ルール

- PCはキーボード入力を基本にする。
- スマホは画面内の数字パッドを基本にする。
- スマホ数字パッドには `π`、`e`、`+`、`-`、`x`、`y`、`√`、`\` を置く。
- PCでは `r` を `√` の代用入力として扱う。
- ルートの中身は `\` で閉じる。例: `r9\` は `√9\` と同じ。
